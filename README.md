# qml-promise

Promise implementation for JavaScript to C++ in Qt QML

## Example

```cpp
Qobject* Test::double(int i)
{
  Promise *promise = new Promise();
  QtConcurrent::run([promise, i] {
    int j = i * 2;
    promise->resolve({i, j});
  });
  return promise;
}
```

```javasript
test.double(5).then(function (i, j) {
  console.log(i, j);
});
```
